# Development_of_approaches_to_implementing_the_concept_of_social_equality_in_pedestrian_zones
Тема исследования: Разработка подходов к реализации концепции социального равенства для пешеходных зон
Автор исследования: Мардисламова Лейсан Ринатовна
Научный руководитель: Чурякова Татьяна Алексеевна
Тема исследования посвящена разработке подходов к реализации концепции социального равенства для пешеходных зон. В работе пешеходные зоны рассматриваются не только как элементы транспортной системы города, но и как часть городской среды, через которую обеспечивается доступ к
объектам повседневного спроса, образовательным учреждениям,общественным пространствам и городским услугам. Именно через пешеходную инфраструктуру реализуется значительная часть повседневной мобильности, поэтому её качество влияет не только на удобство перемещения, но и на возможность участия разных групп населения в городской жизни.
